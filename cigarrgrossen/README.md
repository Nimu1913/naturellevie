# Cigarrgrossen

Svensk grossist för cigarrer & tobak – B2B webbplats.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components
- React Router

## Development

```bash
npm install
npm run dev
```

Opens at `http://localhost:8080`

## Build

```bash
npm run build
npm run preview
```
